# Сценарий демонстрации Energy System

## 1. Подготовка

Открыть два терминала в папке проекта.

В первом терминале запустить API:

```bash
python main.py api
```

Во втором терминале запустить GUI:

```bash
python main.py gui
```

## 2. Вход в систему

1. Ввести логин `admin`.
2. Ввести первичный пароль из переменной окружения или файла `data/initial_admin_credentials.txt`.
3. При первом входе сменить пароль.

## 3. Расчёт режима работы

В главном окне показать блоки параметров:

- нагрузка и оборудование;
- внешние условия;
- дополнительные параметры и оптимизация.

Задать пример:

```text
Общая нагрузка ТЭС: 400 МВт
Количество блоков: 2
Номинальная мощность блока: 300 МВт
Номинальный КПД блока: 0.38
Температура: 25 °C
Влажность: 60 %
Скорость ветра: 3 м/с
Направление ветра: 90 °
Режим работы: Авто
Разрежение в конденсаторе: 88 кПа
```

Нажать `Рассчитать`.

Показать окно результатов:

- нагрузка на энергоблок;
- КПД блока;
- КПД ТЭС брутто;
- собственные нужды;
- КПД ТЭС нетто;
- удельный расход топлива;
- оптимальное количество блоков;
- оптимальное разрежение.

## 4. Графики и аудит расчёта точек

Нажать `Графики + аудит расчета`.

Показать вкладки:

- `Аудит графиков`;
- `Температура`;
- `Блоки`;
- `Конденсатор`.

На вкладке `Аудит графиков` выбрать любую точку и показать:

- исходные данные;
- формулу;
- подстановку чисел;
- поправочные коэффициенты;
- итоговое значение.

Сказать, что график построен по расчётным точкам, а не по произвольным значениям.

## 5. История расчётов

Нажать `История`.

Показать таблицу расчётов и подробный JSON выбранной записи.

## 6. Экспорт

Нажать:

```text
Экспорт CSV
Экспорт PPTX
```

Показать, что файлы сохраняются в папке `exports/`.

## 7. Администрирование

В меню `Администрирование` показать:

- создание пользователя;
- изменение роли пользователя;
- политику безопасности;
- журнал аудита.

В журнале аудита показать события входа, расчёта, API-запросов и административных действий.

## 8. Завершение

Выйти через:

```text
Аккаунт → Выйти
```

Пояснить, что событие выхода фиксируется в журнале аудита.
